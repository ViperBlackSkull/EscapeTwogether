## Getting Started

### Creating or Joining a Room

1. **Start a New Game**
   - Enter your name
   - Click "Create Room"
   - Share the room code with your partner

2. **Join an Existing Game**
   - Enter the room code provided by your partner
   - Enter your name
   - Click "Join Room"

### System Requirements

**Minimum:**
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Stable internet connection
- 2 Mbps download speed

**Recommended:**
- Chrome or Firefox for best performance
- 5 Mbps download speed
- Headphones for audio

## Game Basics

### Objective

Work together with your partner to solve puzzles and escape three mysterious rooms within the time limit!

### Roles

Each player has a unique role that determines what they can see and do:

- **Explorer:** Can interact with room objects and examine details
- **Analyst:** Has access to clues, codes, and puzzle hints

**Important:** You will need to communicate with your partner to solve puzzles, as some information is only visible to one role!

### Rooms

You'll explore three progressively challenging rooms:

1. **The Attic** - Introduction puzzles, get familiar with mechanics
2. **The Clock Tower** - Timing and coordination challenges
3. **The Garden Conservatory** - Complex multi-step puzzles

### Time Limit

You have **45 minutes** to complete all three rooms. The timer is displayed at the top of the screen.

## Controls & Interface

### Desktop Controls

**Mouse:**
- **Left Click:** Interact with objects, select puzzle pieces
- **Drag:** Move puzzle pieces, manipulate objects
- **Scroll:** Zoom in/out (when available)

**Keyboard:**
- **Tab:** Navigate between interactive elements
- **Space:** Activate selected element
- **Escape:** Close modals/puzzles
- **Alt+A:** Open accessibility settings
- **Alt+M:** Skip to main content

### Mobile Controls

**Touch Gestures:**
- **Tap:** Interact with objects
- **Drag:** Move puzzle pieces
- **Pinch:** Zoom in/out
- **Two-finger tap:** Access context menu

### Interface Elements

**Top Bar:**
- Room code (left)
- Timer (center)
- Connection status (right)
- Settings and leave buttons

**Main Area:**
- Game canvas with room view
- Interactive puzzle elements
- Room navigation door

**Sidebar (Desktop):**
- Players panel
- Chat window
- Inventory items
- Hint button

**Bottom Navigation (Mobile):**
- Players, Chat, Items, Hint buttons

## Gameplay Tips

### Communication

- **Use the chat** regularly to share what you see
- **Describe details** - "I see a clock showing 3:15"
- **Ask questions** - "Can you see anything behind the painting?"
- **Confirm actions** - "I'm moving the gear now"

### Puzzle Solving

1. **Explore Everything:** Click on objects to examine them
2. **Take Notes:** Use the chat to save important information
3. **Think Together:** Discuss theories before trying solutions
4. **Use Hints:** If stuck, request a hint (limited uses)
5. **Watch for Clues:** Some puzzles have visual/audio hints

### Role Collaboration

**Explorer Tasks:**
- Examine room objects closely
- Describe what you see to Analyst
- Move puzzle pieces
- Test interactive elements

**Analyst Tasks:**
- Study codes and patterns
- Research clues
- Provide hints to Explorer
- Coordinate overall puzzle strategy

### Time Management

- **Prioritize:** Solve easier puzzles first
- **Don't Stall:** If stuck, move to another puzzle
- **Use Hints:** They're there to help!
- **Stay Calm:** Rushing leads to mistakes

## Puzzle Types

### Matching Puzzles

Match items based on visual characteristics:
- Torn photograph pieces
- Colored gears
- Symbol pairs

### Sequence Puzzles

Enter the correct order:
- Number sequences
- Musical notes
- Color patterns

### Logic Puzzles

Use deductive reasoning:
- Cipher decryption
- Riddles
- Pattern recognition

### Coordination Puzzles

Work together in real-time:
- Simultaneous button presses
- Timing-based challenges
- Information relay tasks

## Troubleshooting

### Connection Issues

**If disconnected:**
1. Wait a moment - automatic reconnection
2. Check your internet connection
3. If not reconnecting, refresh the page
4. You'll rejoin your room automatically

**Lag or Delay:**
- Close other browser tabs
- Check your internet speed
- Try a different browser

### Display Issues

**Game not loading:**
1. Refresh the page
2. Clear browser cache
3. Disable browser extensions
4. Try a different browser

**Text too small/large:**
- Use accessibility settings (Alt+A)
- Adjust browser zoom (Ctrl/Cmd +/â€“)

### Mobile Issues

**Screen not rotating:**
- Check device orientation lock
- Refresh the page

**Touch not responding:**
- Ensure your hands are clean and dry
- Try using a stylus
- Restart the game

### Puzzle Stuck

**Puzzle not responding:**
1. Try clicking other objects first
2. Use a hint if available
3. Leave and return to the puzzle
4. Contact support if issue persists
